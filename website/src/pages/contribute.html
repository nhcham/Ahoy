Everybody can contribute

Establish new language packs

Ahoy! uses many different language packs to enable long messages regardless of the language they're written in (well, not that long, but still longer than the few characters which can be put in a SSID).

If you'd like to help, please have a look at the languages page and check whether the language pack for your language needs feedback from native speakers.

Improve the software

If you'd like to help improve the software, fork the repository here: https://github.com/nhcham/Ahoy.
