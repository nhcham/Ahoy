Everybody can contribute!

Establish new language packs

Ahoy! uses different language packs to enable long messages regardless of the language they're written in (well, not that long, but still longer than the few characters provided by the UTF-8 or UTF-16 encodings).

If you'd like to help, please have a look at the languages page and check whether the language pack for your language needs feedback from native speakers like you!

Improve the software

If you'd like to help improve the software, fork the repository here: https://github.com/nhcham/Ahoy. Feedback is appreciated a lot.
