Languages

While messages consisting of all possible Unicode characters can be sent using Ahoy, the message length is very small by default when UTF-8 or UTF-16 encodings are being used. In order to enable longer messages, specialized language packs can be defined to squeeze the most out of the available space and provide longer messages.

The following languages are already supported or about to be supported if they're not published yet. If your language is marked as draft, please take the time to review it and provide feedback. If your language is missing, write to info@nhcham.org.
